# Trident-G Longevity

This product contains the longevity track based on the Trident-G framework by Mark Ashton Smith - developed in partnership with Alexandra Holgate.

**Aim:** maintenance-first cognitive health support, with optional wearables integration and a full subjective self-report fallback.

## Structure
- `apps/` user-facing apps for maintenance routines
- `assessments/` light-touch checks and adherence probes
- `docs/` product notes, flows, and copy
- `analytics/` event schemas, metrics dictionary, export formats (no raw data)
- `research/` analysis scripts/notebooks (code only, no sensitive datasets)
