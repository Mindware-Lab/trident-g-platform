# Trident-G Resilience

This product contains the resilience track of the Trident-G platform.

**For:** people dealing with burnout, anxiety, low mood, or stress reactivity in high-pressure work environments.

**Aim:** help users stay in a workable mental state under stress, recover quickly from state drift, and maintain consistent training without “heroic” overload.

## Structure
- `apps/` user-facing apps for regulation + training
- `assessments/` brief checks for state, stability, and transfer
- `docs/` product notes, flows, and copy
- `analytics/` event schemas, metrics dictionary, export formats (no raw data)
- `research/` analysis scripts/notebooks (code only, no sensitive datasets)
