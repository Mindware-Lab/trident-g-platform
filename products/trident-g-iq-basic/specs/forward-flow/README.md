# Forward Flow Specs

This folder holds working product specs for the `forward-flow` game family in `trident-g-iq-basic`.

Current structure:

- `docs/`
  - forward-flow game-family spec and implementation plan

This game family is intended to sit alongside the existing active-play families as a separate lane:

- `Forward Flow`
  - for `Locked In` / `in-band` sessions where the player is ready for generative semantic exploration
- `Sustained Attention`
  - for `Spun Out` or unstable sessions where the player needs stabilization first

File conventions:

- `docs/`
  - repo-ready Markdown specs
- `source/`
  - optional copied source files if a later draft originates outside the repo
- `media/`
  - optional extracted or hand-made supporting images
- `mockups/`
  - optional supplied mockup images for this game family
