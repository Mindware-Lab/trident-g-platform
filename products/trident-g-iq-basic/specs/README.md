# Trident G IQ Basic Specs

This folder holds working product specs for `trident-g-iq-basic`.

Current structure:

- `capacity-gym/`
  - game-family spec converted from Word to Markdown
  - player-feedback spec converted from Word to Markdown
  - supplied n-back mockup images
- `forward-flow/`
  - forward-flow game-family spec and implementation plan
- `zone-coach/`
  - zone-coach spec converted from Word to Markdown
- `shared-systems/`
  - telemetry and game-map spec converted from Word to Markdown
- `economy/`
  - credit-economy spec converted from Word to Markdown

File conventions:

- `docs/`
  - repo-ready Markdown versions of the specs
- `source/`
  - copied source `.docx` files for traceability
- `media/`
  - images extracted during Word-to-Markdown conversion
- `mockups/`
  - manually supplied mockup images that belong with a spec area

Add future spec uploads to the relevant domain folder rather than creating loose files at the top level.
