# Capacity Gym Specs

This folder groups current Capacity Gym specifications and mockup assets.

Contents:

- `docs/capacity-gym-games.md`
  - converted game-family spec
- `source/capacity-gym-games.docx`
  - copied source document
- `player-feedback/`
  - converted player-feedback spec, source file, and extracted media
- `mockups/n-back/`
  - supplied mockup images for relational, emotional, interference, AND, and XOR n-back variants
- `media/`
  - images extracted from the main games document during conversion

Use this folder for additional Capacity Gym docs such as new rules, variants, telemetry notes, or scoring details.
