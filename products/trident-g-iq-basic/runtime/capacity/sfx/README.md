# Capacity Sandbox SFX

Drop optional sound files in this directory using:

`cg_sfx_<event_id>_<variant>.wav`

Example:

`cg_sfx_trial_hit_1.wav`

If assets are missing, the sandbox will fall back to synthesized SFX.

## Event IDs

- `ui_tap_soft`
- `game_match_press`
- `trial_hit`
- `trial_false_alarm`
- `trial_miss`
- `pause_on`
- `resume_on`
- `session_stop_discard`
- `block_complete_neutral`
- `n_level_up`
- `n_level_down`
