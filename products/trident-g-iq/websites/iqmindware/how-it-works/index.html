Transfer protocol

Two kinds of progression

The programme uses horizontal and vertical transfer together so training does not become tied to one familiar task format.

H
Horizontal transfer
Same skill, different surface forms
Built in
What it means
The coach practises the same underlying cognitive demand across different stimulus types — letters, locations, objects, relations, real-world meanings, nonsense meanings.
Why it matters
Stops the skill being tied to one familiar game format. Forces genuine generalisation rather than surface-level familiarity.
View protocol
V
Vertical transfer
Increasing reasoning demand
Built in
What it means
The coach moves from easier recognition and control tasks toward deeper relational binding, rule use, inference, and fluid reasoning challenges as training progresses.
Why it matters
Capacity Gym prepares the control system. Reasoning Gym then demands you use that control for rule-based thinking and logical inference — the higher-order target.

Modules

What each module does

Four modules in one app. Each has a specific role in the programme.

1
Zone Pulse
Daily cognitive control check
Live
What it measures
Cognitive control capacity in bits per second, using the Majority Function Task — a short arrow-direction probe.
When to use
Before each daily training session. Recommended but skippable.
Open Zone Pulse
2
Capacity Gym
Working memory and attention control
Live
What it trains
Working memory, attention control, and distraction resistance under controlled pressure. Uses N-back tasks including novel relational N-back variants.
When to use
Each training day, following Zone Pulse, as directed by the coach.
Open Capacity Gym
3
Reasoning Gym
Inference and fluid reasoning
Live
What it trains
Relational binding, rule use, constraint solving, and must-follow inference. Uses both real-world and nonsense forms so the skill is not surface-dependent.
When to use
Each training day, combined with Capacity Gym, as directed by the coach.
Open Reasoning Gym
4
Tracker
Baseline, trend, and progress evidence
Live
What it tracks
SgS-12 for fluid reasoning snapshots (pre and post). Psi-CBS for applied cognition, resilience, and AI-use effects over time.
When to use
SgS-12 Pre before the programme. Psi-CBS every 5 sessions. SgS-12 Post after the programme.
