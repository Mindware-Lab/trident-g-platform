## A practical cognition stack (new release)

### Zone Coach — readiness + fast re-entry
A short daily check you run before training or deep work. It helps you decide: **push, maintain, or reset first** — so you don’t grind on the wrong day.

### Capacity Gym — cognitive control training (working memory + attention + relational reasoning)
A gamified training gym with a coach layer designed to reduce rote learning and improve stability. You train both core capacity and reasoning wrappers, with clear progress signals.
