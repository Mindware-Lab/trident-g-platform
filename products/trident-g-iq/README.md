# Trident-G IQ

This product contains the IQ track of the Trident-G platform.

**Aim:** improve performance under load and support far transfer via (1) state gating and re-entry, (2) capacity training, and (3) mindware/operator practice.

## Structure
- `apps/` user-facing apps for training and coaching
- `assessments/` baseline/post checks and micro-probes
- `docs/` product notes, flows, and copy
- `analytics/` event schemas, metrics dictionary, export formats (no raw data)
- `research/` analysis scripts/notebooks (code only, no sensitive datasets)
