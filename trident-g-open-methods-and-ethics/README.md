# Trident-G Open Methods and Ethics

Public methods, ethics, schemas, and aggregate publication artefacts for Trident-G IQ. No runtime app code, no private infrastructure, no user-level data.

## What this repo publishes

- Protocol publications for IQ tools:
  - Zone Coach
  - Capacity Gym
  - G Tracker battery
- Governance documents for ethics, claims, AI use, and evidence policy
- Data publication policy and public aggregate schemas
- Templates for aggregate releases and manifests

## What this repo does not publish

- Runtime app source code or private infrastructure
- Anti-gaming implementation constants
- User-level data or identifiable personal information

## Open Methods, Protected Execution

Open:
- Constructs and training/measurement intent
- Progression/validation logic families in plain language
- Metric definitions, claims limits, privacy rules, and publication schemas

Protected:
- Runtime source code and internal deployment architecture
- Anti-abuse and anti-gaming heuristics
- Private operational thresholds and proprietary execution playbooks
